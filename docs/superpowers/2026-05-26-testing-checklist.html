  • Permissions — roles owner/admin/member; requireRole gates team-admin/settings/integrations/custom-field mutations; assertCanWrite enforces "members edit/delete only records they own" on contacts/companies/deals/activities (incl. deal move). owner_user_id set on create + backfilled (migration 0011); CHECK constraint on org_members.role (0012).
