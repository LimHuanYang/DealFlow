One EngineMailer key — how the "From" address works

Both use a single shared DealFlow EngineMailer account. The difference is the visible From address your recipient sees.

⚡ The rule: you can only send from a domain that's verified in the EngineMailer account. With one shared key, that's your domain by default — so Option A puts your domain in the From line (replies still go to the user via Reply-To), while Option B lets each org put their own domain in the From line, but only after they add DNS records.
